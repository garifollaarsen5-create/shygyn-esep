# ШығынЕсеп 💰

Жеке шығындарды есептейтін PWA (телефонға орнатылатын қосымша).
Деректер тек сіздің құрылғыңызда сақталады — сервер де, тіркелу де керек емес.

## Мүмкіндіктер

- ➕ Шығын қосу / өзгерту / жою (сома, категория, күн, ескертпе)
- 🗂 Категориялар (өзіңіз қосуға/өшіруге болады)
- 📊 Статистика: категория бойынша (donut) және айлар бойынша (bar)
- 🎯 Айлық бюджет пен асып кету ескертуі
- 🌐 Тіл: қазақша / орысша (ауыстырмалы)
- 💱 Валюта: ₸ теңге / ₽ рубль / $ доллар / € евро
- 🌙 Күңгірт / ☀️ жарық тақырып
- 📱 PWA — телефон экранына орнатылады, интернетсіз де істейді
- ⬇️ Деректерді JSON-ға сақтау (export)

## Іске қосу (компьютерде сынау)

```bash
npm install      # кітапханаларды орнату (бір рет)
npm run dev      # әзірлеу режимі → http://localhost:5173
npm test         # логика тесттері
npm run build    # production жинақ (dist/ қалтасына)
npm run preview  # жиналған нұсқаны қарау
```

## 📲 Телефонға қалай орнатамын?

PWA болғандықтан, оны интернетке жариялап, телефоннан ашу керек. Ең оңай 2 жол:

### А нұсқа — тегін хостингке жариялау (ұсынылады)

1. `npm run build` → `dist/` қалтасы жасалады.
2. `dist/` ішіндегісін тегін хостингке жүктеңіз:
   - **Netlify** (netlify.com) — `dist` қалтасын браузерге сүйреп апару жеткілікті, немесе
   - **Vercel** / **GitHub Pages** / **Cloudflare Pages**.
3. Алынған сілтемені (мыс. `https://...netlify.app`) телефон браузерінен ашыңыз.
4. **Android (Chrome):** мәзір (⋮) → «Қосымшаны орнату» / «Add to Home screen».
   **iPhone (Safari):** Бөлісу (⬆️) → «Главный экранға қосу» / «Add to Home Screen».
5. Дайын — телефон экранында өз иконкасымен қосымша пайда болады.

### Б нұсқа — Wi-Fi арқылы үйде сынау

```bash
npm run dev -- --host
```

Терминалдағы `Network:` сілтемесін (мыс. `http://192.168.x.x:5173`) бір Wi-Fi-дағы
телефоннан ашыңыз. (Бұл жол толық орнату/офлайн үшін HTTPS керек ететінін ескеріңіз —
тұрақты қолдану үшін А нұсқасы дұрыс.)

## Технология

React + TypeScript + Vite + Tailwind CSS v4 + vite-plugin-pwa.
Диаграммалар — тәуелсіздіксіз, қолдан жазылған жеңіл SVG.
Дерек — браузердің `localStorage`-інде.
